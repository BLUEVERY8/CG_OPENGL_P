모델좌표 : 각각의 물체의 중심, 물체별로 부여
전역좌표 : Default = 그래픽창 중심
시점좌표 : 바라보는 시점에 따라 바뀌는 좌표
절단좌표 : 시점으로부터 보이지 않는 물체
정규좌표 : Default = 그래픽창 중심 glortho(left, right, bottom, top, nearVal, farVal) : 보여지는 범위 설정
화면 좌표 : 모니터에 따라 달라지는 좌표 중심값 변경 불가, 좌측 상단이 중심

뷰포트 : 그래픽창에 보여주고 싶은 영역 Default = 전체, (0, 0) = 좌측 하단 glViewport(시작점(x, y), 너비, 높이)
왜곡 방지 : 가로 세로의 비율을 같게 하는 것

왜곡 방지 reshape 
glViewport()
GLfloat WidthFactor = NewWidth / 윈도우 너비
GLfloat HeightFactor = NewHeight / 윈도우높이

glMatrixMode(GL_PROJECTION); glLoadIdentity();
